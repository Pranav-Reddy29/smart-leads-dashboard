# Smart Leads Dashboard

A production-ready MERN Stack CRM dashboard built with multi-tenant architecture, authentication, lead management, analytics, and modern UI.

## Live Demo

### Frontend
https://smart-leads-dashboard-p.vercel.app/

### Backend API
https://smart-leads-dashboard-e8fu.onrender.com/api

---

# Features

## Authentication & Authorization
- JWT Authentication
- Secure Login & Registration
- Protected Routes
- Persistent Authentication
- Logout Functionality

## Dashboard Analytics
- Dynamic Lead Statistics
- Total Leads Count
- Qualified Leads
- Contacted Leads
- Lost Leads

## Lead Management
- Create Lead
- Edit Lead
- Delete Lead
- Lead Details Page
- Lead Status Management
- Lead Source Tracking
- Notes & Follow-ups

## Multi-Tenant Architecture
- Organization-based data isolation
- Tenant-aware APIs
- Scalable SaaS-ready structure

## Modern Frontend
- React + TypeScript
- Responsive UI
- Sidebar Navigation
- Clean Dashboard Design
- Axios API Integration
- Zustand State Management

## Backend Features
- RESTful API Architecture
- MongoDB + Mongoose
- Express.js Middleware
- Error Handling
- Request Validation
- Environment-based Config

## Production Features
- MongoDB Atlas Integration
- Render Backend Deployment
- Vercel Frontend Deployment
- Environment Variables
- CORS Configuration
- TypeScript Support
